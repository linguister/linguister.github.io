# Linguister

Página web de presentación para dos proyectos de creación lingüística: **Cuchilleras** y **FLIPAR**.

## Descripción

Esta página web sirve como punto de entrada para explorar dos proyectos relacionados con la escritura y la representación fonética:

### Cuchilleras
La fuente que el español merece. Una tipografía diseñada para adaptarse al español y sentar las bases de su futura forma escrita.

### FLIPAR
Representación lógica del _AFI_ (_Alfabeto Fonético Internacional_). Un sistema optimizado para la representación fonética que facilita la transcripción y el análisis lingüístico de manera clara y precisa a través de la lógica.
